# Risk Matrix

變更風險分級與處置。

| 等級 | 定義 | 範例 | 處置 |
|---|---|---|---|
| **P0** | Production 中斷 | 推播全停、workflow 失效、scheduler 壞 | 立即停止，最高優先修復 |
| **P1** | 錯誤交易訊號 / 資料污染 | 選股邏輯錯、label 錯、history 污染 | 禁止 Merge，退回重做 |
| **P2** | 非核心 / 文件影響 | README 不同步、log 格式改變 | 允許，走簡化流程 |
| **P3** | 純文件 | 錯字、註解、排版 | 直接 Merge |

## 判定原則

- 不確定等級時，就高不就低。
- P0/P1 一律附證據，並標明影響模組。
