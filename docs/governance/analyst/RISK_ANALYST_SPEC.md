# Risk Analyst — 規格（第一版啟用）

## 狀態
✅ **啟用（有限資料）**。用現有價量，不接籌碼/事件。

## 輸入
| 來源 | 用途 |
|---|---|
| 價格歷史（market_data 現有） | 計算波動率（標準差） |
| `pred_gain` vs `STOP_THRESHOLD` | 風報比估算 |
| `chg`, `vx` | 短線過熱/爆量風險 |

## 評分邏輯（Risk Score 0–100，分數越高越安全）
- 波動率越高 → 分數越低
- 風報比（pred_gain / 停損幅度）越高 → 分數越高
- 當日爆量 + 大漲 → 過熱風險，扣分

## 輸出
```json
{ "score": 55, "verdict": "WATCH",
  "reasons": ["近期波動上升", "風報比中等"],
  "risks": ["短線過熱", "波動放大"] }
```

## 明確缺失（不捏造）
| 需要 | 狀態 |
|---|---|
| 籌碼（法人/融資券） | ❌ 無資料，不評 |
| 事件風險（財報日/除權息） | ❌ 無資料，不評 |
本分析師輸出須註明「僅基於量價，未含籌碼與事件」。
