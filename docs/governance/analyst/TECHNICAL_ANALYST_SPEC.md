# Technical Analyst — 規格（第一版啟用）

## 狀態
✅ **啟用**。三位裡唯一具備完整真實資料者。

## 輸入（皆為候選股現有欄位，不新增資料源）
| 欄位 | 意義 |
|---|---|
| `rsi` | RSI 相對強弱 |
| `ma20_diff` | 距 MA20 百分比（趨勢） |
| `vx` | 量比（當日量/20日均量，動能） |
| `chg` | 當日漲跌% |
| `hi_lo_pos` | K 棒收盤位置 |

## 評分邏輯（Technical Score 0–100）
以現有欄位組合，示意規則（實作時定稿）：
- 趨勢：`ma20_diff > 0` 加分（站上均線）
- 動能：`vx` 適中偏高加分；過熱（極端量）扣分
- RSI：50–70 健康加分；>80 過熱扣分；<30 超賣提示
- 收盤強度：`hi_lo_pos` 偏高加分

## 輸出
```json
{ "score": 72, "verdict": "BUY|WATCH|REJECT",
  "reasons": ["站上MA20", "RSI健康區", "量能溫和放大"] }
```
